# css入门

## 盒模型

- 尺寸
  - `width` / `height`：宽/高
  - `padding`：内边距
  - `margin`：外边距
  - `border`：边框
  - `box-sizing`：如何计算盒子尺寸
- 背景与边框
  - `background-color`：背景颜色
  - `background-image`：背景图
  - `background-size`：背景尺寸
  - `background-position`：背景位置
  - `background-repeat`：是否重复
  - `border-radius`：圆角
  - `box-shadow`：盒子阴影
  - `border`：边框样式
